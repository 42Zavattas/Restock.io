Introduction

The main purpose of Restock is to provide a ready-to-use toolbox to mock your REST calls and don't bother with them. You can create some simple results composed of one item of the following supported types, or more complex one by nesting them inside an array or an object.

Feel free to contact us when you encounter an issue or an inconsistency.

List of codes

Basic
s / S Any string ajeke
n / N Any number 32 / -491264923402240
b "Any" boolean true
d Any date 2054-06-04T01:37:33.311Z
Extended
e Any email kqzdp@aiklqq.gv
p Any phone (US formatted) (561) 927-1354
a Any address (US formatted) 1337 Netne Le

c Any character b

Practice cases

cURL 
curl http://restock.io/api/5s
AngularJs 
$http.get('http://restock.io/api/5s').then(function (res) {
  console.log(res);
});
NodeJs 
var options = {
  host: 'http://restock.io',
  port: 80,
  path: '/api/5s',
  method: 'GET'
};

http.request(options, function (res) {
  res.on('data', function (data) {
    console.log(data);
  });
})
Python 
import urllib

url = 'http://restock.io/api/5s'
u = urllib.urlopen(url)
data = u.read()
PHP 
$response = file_get_contents('http://restock.io/api/5s');
$data = json_decode($response);
